# Angular App

`npm install -g angular-generator`

### How to Use

```
npm install
node app.js
```

#### Guide

```html
To add an icon:
<i
  class="fa fa-american-sign-language-interpreting fa-5x"
  aria-hidden="true"
></i>
```
